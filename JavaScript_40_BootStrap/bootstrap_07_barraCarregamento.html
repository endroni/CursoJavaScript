Barra de Progressão

pode ser usada para mostrar ao usuário o quanto ele está em um processo.

.progress uma classe pai

.progress-bar uma classe filho

.width propriedade do css para definir o progresso

.height propriedade do css para definir a altura do progresso

Etiqueta na barra de progresso

.progress-bar-striped barra listrada

.progress-bar-animated barra animada
