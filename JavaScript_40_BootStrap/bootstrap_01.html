Bootstrap 3 vs Bootstrap 4

Bootstrap 4 é a versão mais recente do Bootstrap; com novos componentes, folha de estilo mais rápida e mais capacidade de resposta.

Se você precisar de suporte para o IE8-9 ou anteriores, use o Bootstrap 3.
