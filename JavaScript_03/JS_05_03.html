A sintaxe JavaScript é o conjunto de regras, como os programas JavaScript são construídos:

Valores literais

Os valores literais, são os valores fixos

Os números são escritos sem aspas.

Strings são textos, escritos entre aspas duplas ou simples.

Valores variáveis

O JS utiliza var, const ou let como palavra-chave para declarar variáveis

let e const (2015)

A versão 2015 do JavaScript(ES6 - ECMAScript 2015) permite o uso da const para definir variável que não pode ser reatribuída.

A let para definir variável com escopo restrito.

Identificadores são nomes

Identificadores são usados ​​para nomear variáveis ​​(e palavras-chave, funções e rótulos).

O primeiro caractere deve ser uma letra, um sublinhado (_) ou um cifrão ($).

Números não são permitidos como primeiro caractere.

Hífens não são permitido

Case Sensitive
